when looking into our stm32-based project, I see the following way to implement cmsis:

use NXP HAL layer (already available)
use FreeRtos (already available)
add cmsis files (e.g. get the from stm32-based project, available via STM)
used files will be cmsis_os.c, cmsis_os.h, core_cmFunc.h, cmsis_gcc.h
adapt cmsis_gcc.h to NXP processor (only a few assembly functions)
I do not see a copyright problem with this solution. The copyright is always mentioned in the source files.
